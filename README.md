## Features

- Flutter (Dart)
- State Management (Bloc, Cubit, Provider)
- Firebase Auth
- Secure Storage
- SharedPreference
- Model -> Repository -> State Management -> View  
- Biometrics Auth (Not supported by firebase --> Hack!)
